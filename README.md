# Finance List API

Finance List API is a simple RESTful API built with Express.js for managing financial records. This project provides basic CRUD functionality, allowing users to create, read, update, and delete finance data such as title, nominal amount, money flow type, date, and currency.

## Features

- Get all finance records
- Add a new finance record
- Update an existing finance record
- Delete a finance record
- JSON-based request and response
- Simple in-memory data storage

## Tech Stack

- Node.js
- Express.js
- JavaScript
